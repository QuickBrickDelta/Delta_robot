# Politique de Sécurité

## Signaler une vulnérabilité

Si vous découvrez une vulnérabilité de sécurité dans ce projet, veuillez envoyer un e-mail aux mainteneurs (voir le README pour les informations de contact) ou ouvrir une issue privée si la plateforme le permet.

Toutes les vulnérabilités de sécurité seront traitées rapidement.

## Versions supportées

Actuellement, seule la branche `main` est activement supportée pour les mises à jour de sécurité.
